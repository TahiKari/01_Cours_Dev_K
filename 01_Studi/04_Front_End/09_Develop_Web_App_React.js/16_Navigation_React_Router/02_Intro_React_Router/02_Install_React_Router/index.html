Installation de React Router

Les différentes manières d'installer React Router

Il existe plusieurs façons d'installer React Router, chacune avec ses propres avantages et inconvénients. Voici les différentes manières de l’installer :

La première méthode d'installation consiste à utiliser le gestionnaire de paquets npm. C'est la façon la plus courante d'installer React Router. Pour ce faire, il suffit d'exécuter la commande "npm install react-router-dom" dans le terminal de votre projet. Cette commande va installer la dernière version de React Router dans votre projet et ajouter une dépendance dans votre package.json.

Une deuxième méthode consiste à utiliser le gestionnaire de paquets Yarn. Si vous préférez utiliser Yarn, vous pouvez exécuter la commande "yarn add react-router-dom" pour installer React Router dans votre projet. Cette méthode est similaire à l'installation via npm, mais utilise le gestionnaire de paquets Yarn.

Enfin, une troisième méthode consiste à installer React Router en utilisant un CDN. Si vous ne souhaitez pas utiliser npm ou Yarn, cette méthode peut être une bonne alternative. Il suffit d'inclure les liens CDN dans la section head de votre fichier HTML et de les utiliser dans votre application. Cette méthode peut être plus rapide pour les petites applications, mais peut poser des problèmes de compatibilité avec certaines versions de React Router.

Création d’un projet React

Méthode

Nous allons créer un projet React de base via CRA en utilisant la commande suivante dans le terminal de notre éditeur de code :

npx create-react-app mon-projet

Nous avons maintenant un template de base, fourni par CRA. Nous pouvons accéder au répertoire de notre application en tapant la commande suivante dans le terminal :

cd mon-projet

Installer React Router avec npm

Méthode

Nous allons maintenant installer React Router en tant que dépendance de notre application, nous pouvons utiliser la commande suivante dans le terminal de notre éditeur :

npm i react-router-dom

Nous avons maintenant React Router installé en tant que dépendance sous sa dernière version 6.10.0.

Lancer le serveur de développement

Méthode

Maintenant que nous avons installé React Router, nous pouvons lancer le serveur de développement en utilisant la commande suivante :

npm start
