Contexte

React Router est une bibliothèque qui permet de gérer la navigation dans une application web basée sur React. Que vous construisiez une application web classique ou une Single-Page Application (SPA), React Router offre des fonctionnalités avancées pour la navigation.

En premier lieu, nous aborderons les notions de base de React Router dans sa version 6.10.0, y compris son installation et son utilité pour la navigation dans une application web. Nous allons également découvrir les avantages de l'utilisation de React Router pour les SPA, et comment cette bibliothèque répond aux besoins de navigation dans ces applications.

Ensuite, nous verrons plus en profondeur les fonctions de navigation de React Router. Nous découvrirons comment utiliser les composants de base de ce dernier pour mettre en place une navigation, ainsi que les paramètres de route et les redirections.
