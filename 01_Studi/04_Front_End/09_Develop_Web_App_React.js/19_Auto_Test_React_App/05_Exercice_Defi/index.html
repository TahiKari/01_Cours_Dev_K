Exercice défi

Imaginez avoir le besoin de réaliser un test avec Jest qui vérifie que le nombre d’utilisateurs initiaux présents dans le tableau users est inférieur à 5, et vérifier que la valeur totale soit bien égale à 4.

Question

Effectuez une recherche sur les références des comparateurs afin de trouver celui qui permet de vérifier si un nombre est inférieur à un autre, et celui qui permet de vérifier la longueur exacte de notre tableau.

Solution

Dans la documentation de Jest, l’ensemble des comparateurs est référencé ici : https://jestjs.io/fr/docs/expect.

Après recherche, nous avons pu identifier que le comparateur utilisé pour vérifier qu’un nombre est inférieur à un autre est toBeLessThan() : https://jestjs.io/fr/docs/expect#tobelessthannumber--bigint.

Ensuite, pour vérifier la longueur de notre tableau, nous devons utiliser toHaveLength() : https://jestjs.io/fr/docs/expect#tohavelengthnumber.
