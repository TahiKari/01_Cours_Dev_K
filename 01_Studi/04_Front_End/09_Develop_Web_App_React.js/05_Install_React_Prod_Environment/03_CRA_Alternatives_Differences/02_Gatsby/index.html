Gatsby

Présentation de Gatsby

Gatsby est un framework de développement web statique qui est basé sur React. Il est conçu pour faciliter la création d'applications web rapides et performantes en utilisant des données provenant de différentes sources.

L'objectif principal de Gatsby est de simplifier le processus de création de sites web en utilisant des fichiers statiques, plutôt que des bases de données dynamiques. Cela permet de créer des sites web plus rapides, plus faciles à déployer et plus sécurisés.

En utilisant Gatsby, vous pouvez facilement créer des pages web avec du contenu dynamique en utilisant les données provenant de sources telles que des fichiers JSON, des API, des CMS, etc. De plus, Gatsby utilise GraphQL, un langage de requête pour récupérer et manipuler les données, ce qui facilite encore plus le processus de développement.

Gatsby est également connu pour sa grande flexibilité et sa modularité. Il dispose d'une vaste bibliothèque de plugins qui vous permettent d'ajouter des fonctionnalités spécifiques à votre application. Vous pouvez également personnaliser l'apparence de votre application en utilisant des thèmes ou en créant votre propre système de conception.

En somme, Gatsby est un excellent choix pour créer des applications web rapides, performantes et sécurisées. Il offre une grande flexibilité et une large gamme de fonctionnalités qui vous permettent de créer des applications personnalisées pour répondre aux besoins de votre entreprise ou de vos utilisateurs.

Initier un projet React avec Gatsby

Méthode

Nous allons maintenant découvrir comment créer un projet React avec Gatsby, ouvrez un nouveau terminal et tapez la commande suivante :

npm init gatsby -y

Spécifier le nom du projet

Méthode

Il nous est maintenant demandé de donner un nom à notre application, dans le contexte du cours nous allons appeler notre projet 'mon-projet'.

Appuyez sur 'Entrée' après avoir taper le nom de notre application dans le terminal.

Ensuite, il nous est demandé de nommer le dossier dans lequel sera créée notre application, par défaut le nom spécifié est le même que le nom de l’application que nous avons indiqué précédemment. Nous allons garder ce nom et appuyer sur 'Entrée' pour valider notre choix.

Préférences du projet

Méthode

Tout comme avec Next.js, Gatsby nous propose d’utiliser TypeScript ou JavaScript pour notre projet, par défaut le curseur de sélection est déjà sur JavaScript. Dans ce cas nous pouvons appuyer sur 'Entrée' pour valider notre choix.

Après quoi, nous avons la possibilité d’utiliser un CMS tels que WordPress ou Shopify. Par défaut, le curseur est sur 'No', nous pouvons donc appuyer sur 'Entrée' pour valider notre choix. Il est possible d’utiliser un CMS plus tard dans notre projet, même si nous avons spécifié 'No' lors de l’initialisation de celui-ci.

Il nous est ensuite demandé si l'on souhaite utiliser un système de style tels que PostCSS ou Sass. Les systèmes de style sont utilisés pour gérer et organiser les styles CSS, afin de maintenir la cohérence et la qualité de la présentation visuelle d'un projet. Dans le cas de notre projet, nous allons choisir 'No' et appuyer sur 'Entrée' pour valider notre choix. Il est également possible d’utiliser un système de style plus tard dans notre projet, même si nous avons spécifié 'No' lors de l’initialisation de celui-ci.

Fonctionnalités additionnelles

Méthode

Enfin, nous avons la possibilité d’ajouter des fonctionnalités additionnelles, telles que les tags 'gtag', liées à Google Analytics ou encore la génération d’un fichier 'Manifest'

Pour implémenter l’une de ces fonctionnalités dans notre projet, il suffit de sélectionner la fonctionnalité en question avec le curseur et d’appuyer sur 'Espace' pour cocher celle-ci.

Dans le contexte du cours, nous n'ajoutons aucune fonctionnalité supplémentaire à notre projet, nous pouvons donc sélectionner 'Done' avec le curseur et appuyer sur 'Entrée' pour confirmer nos choix.

Confirmation de la création du projet

Méthode

Une demande de confirmation apparaît maintenant, nous pouvons maintenant appuyer sur 'Entrée' ou encore 'y' pour valider la création du projet.

Lancer l’application

Méthode

Notre application est maintenant créée avec Gatsby, nous allons accéder au dossier créé en utilisant la commande suivante :

cd mon-projet

Ici 'mon-projet' correspond au nom du projet que nous avons défini précédemment.

Ensuite nous allons utiliser la commande suivante pour lancer notre application :

npm run develop

Cela va lancer votre application sur le port 8000. Vous pouvez accéder à votre application en ouvrant votre navigateur et en visitant l'URL http://localhost:8000.

Vous venez de créer une base d'application React en utilisant Gatsby. Vous pouvez maintenant ajouter des fonctionnalités à votre application en créant des pages et en ajoutant des composants React.
