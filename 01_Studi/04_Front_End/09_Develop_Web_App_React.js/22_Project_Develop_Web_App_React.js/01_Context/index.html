Contexte

React.js est une bibliothèque JavaScript populaire utilisée pour développer des applications web interactives et réactives. Il offre une approche basée sur les composants, ce qui signifie que vous pouvez diviser votre application en plusieurs composants réutilisables et les composer pour créer une interface utilisateur complexe.

Au terme de ce cours, vous aurez mis en place une application complètement opérationnelle, permettant à l’utilisateur de saisir le nom de n’importe quelle ville à travers le monde, et de s’informer en temps réel sur l’heure et la météo de cette ville.
