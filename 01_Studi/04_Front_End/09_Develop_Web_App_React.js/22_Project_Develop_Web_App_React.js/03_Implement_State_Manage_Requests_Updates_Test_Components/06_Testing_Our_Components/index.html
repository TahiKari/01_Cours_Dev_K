Lancer les tests

Méthode

Nous pouvons lancer les tests de notre application afin de nous assurer qu’il s’exécutent tous avec succès. Pour cela, ouvrez le terminal et accédez au dossier de l’application via la commande suivante :

cd mon-projet

Enfin, nous pouvons écrire la commande suivante pour lancer les tests :

npm run test

Si l’ensemble des tests sont passés, nous avons maintenant l’assurance que notre application réagit correctement par rapport à nos attentes initiales.
