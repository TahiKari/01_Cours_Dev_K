L'initialisation de l’application avec CRA

Dans ce projet, nous allons créer une application météo avec React. Le développement de cette application nous permettra d’aborder la mise en place d’une architecture complète React. Nous allons réaliser des appels vers des API avec Axios afin de récupérer des données que nous afficherons dans nos composants stylisés avec Styled-Components. Nous utiliserons également Redux pour gérer l’état global de notre application, et Jest pour la tester.

Initialiser avec CRA

Méthode

Nous devons initialiser notre projet en utilisant CRA. Pour effectuer cela, créez un dossier auquel vous accédez avec votre éditeur de code.

Ensuite, dans le terminal de votre éditeur, écrivez la commande suivante afin d’initier le projet :

npx create-react-app mon-projet
