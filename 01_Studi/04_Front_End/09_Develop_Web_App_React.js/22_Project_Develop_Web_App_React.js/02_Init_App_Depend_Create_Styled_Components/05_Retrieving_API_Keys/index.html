La récupération des clés d’API

L'utilisation d’API

Remarque

Nous allons utiliser 2 API dans ce projet : OpenWeatherMap afin d’accéder aux données météorologiques d’une ville en temps réel, ainsi que TimeZoneDb qui va nous permettre de récupérer la date et l’heure en temps réel de cette ville.

Nous avons besoin d’avoir l’autorisation d’utiliser ces API et pour cela, nous devons récupérer une clé que nous allons utiliser dans le code de notre application.

Afin de récupérer ces clés, vous devez créer un compte sur chacune des plateformes accessibles aux liens suivants :

  • Open weather map
  • Time zone db

Stocker des clés dans un fichier

Méthode

Maintenant que nous avons récupéré les clés d’accès pour ces API, nous pouvons créer un fichier « .env » dans lequel nous allons stocker ces clés dans des variables d’environnement afin de les utiliser plus tard dans l’application. Inscrivez le code suivant dans le fichier créé :

REACT_APP_WEATHER_KEY = 'votre-clé-récupérée-sur-openweathermap' REACT_APP_TIMEDB_KEY = 'votre-clé-récupérée-sur-timezonedb'
