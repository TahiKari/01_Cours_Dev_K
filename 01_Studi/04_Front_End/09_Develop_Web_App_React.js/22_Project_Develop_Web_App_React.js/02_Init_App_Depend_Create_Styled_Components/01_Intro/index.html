Introduction

Les objectifs

  • Initialiser l’application en utilisant Create React App (CRA), en créant un dossier et en exécutant la commande npx create-react-app mon-projet.
  • Installer les dépendances nécessaires à la création du projet, notamment styled-components, redux toolkit, react-redux, axios et Jest. Cela peut être fait en accédant au dossier du projet avec la commande cd mon-projet et en exécutant la commande npm install styled-components@v5.3.11 @reduxjs/toolkit react-redux axios.
  • Créer des composants stylisés pour l’application. Dans cette partie, 3 composants sont créés : Header, Meteo et CityInput. Chaque composant est créé dans un dossier correspondant et utilise styled-components pour le styliser.
  • Ajouter une couleur de fond à l’application en utilisant un dégradé linéaire dans le fichier « index.css ».
  • Stocker les clés d’API nécessaires pour accéder aux données météorologiques en temps réel et à la date/heure en temps réel. Les clés sont stockées dans des variables d’environnement dans un fichier « .env » pour une utilisation ultérieure dans l’application.
