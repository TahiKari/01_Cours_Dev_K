L'installation des dépendances

Installer les dépendances au projet

Maintenant que notre projet est initié, nous pouvons installer les dépendances nécessaires à la création de notre projet. Nous aurons besoin d’installer Styled-Components afin de styliser nos composants, Redux toolkit et React-redux afin de gérer l’état global de notre projet, Axios pour effectuer des requêtes vers les API et Jest, qui est déjà livré avec CRA pour tester nos composants.

Méthode

Ouvrez le terminal de l’éditeur et accédez au dossier de votre projet en écrivant la commande suivante :

cd mon-projet

Ensuite, écrivez la commande suivante afin d’installer les différentes dépendances du projet :

npm install styled-components@v5.3.11 @reduxjs/toolkit react-redux axios

Les versions utilisées

Remarque

Voici les versions respectives de chaque dépendance utilisée dans ce projet :

  • React : 18.2.0
  • Jest : 5.16.5
  • Redux Toolkit : 1.9.5
  • React Redux : 8.0.7
  • Axios : 1.4.0
