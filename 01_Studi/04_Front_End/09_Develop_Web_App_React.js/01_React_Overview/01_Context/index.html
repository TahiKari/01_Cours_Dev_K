Contexte

Avec l'accroissement des performances de nos appareils personnels d'année en année, de plus en plus de traitements qui étaient jusqu'alors réservés au serveur (plus puissant) sont désormais déportés sur nos ordinateurs, tablettes et mobiles. Faisant ses débuts comme un langage de scripts relativement limité, JavaScript est aujourd'hui devenu un puissant outil qui permet aux développeurs de réaliser des applications et des interfaces utilisateurs entièrement exécutées par le navigateur. React est un framework JavaScript permettant de construire des applications avec un rendu côté client simple et intuitif.
