exercice défi

Imaginez avoir besoin d’un nouveau projet dans Firebase afin de gérer l’authentification des utilisateurs dans votre nouvelle application. Nous devons initier un nouveau projet et activer le service d’authentification qui va permettre aux utilisateurs de se connecter via une adresse e-mail ainsi qu’un mot de passe.

Question

Créez un nouveau projet Firebase.

Solution

Pour initier un nouveau projet Firebase, nous pouvons nous rendre sur l’URL suivante et appuyer sur « getStarted » afin d’accéder à la page qui contient nos projets existants : Firebase

En arrivant sur cette page, nous avons la possibilité de cliquer sur le bouton « Ajouter un projet », ce qui nous redirigera vers la page permettant de saisir le nom de ce nouveau projet.

Après avoir saisi le nom du projet et fait notre choix concernant l’utilisation de Google Analytics dans notre projet, l’initialisation de notre projet a maintenant débuté

Question

Activez le service d’authentification dans le projet.

Solution

Maintenant que notre projet est créé, nous pouvons cliquer sur le bouton « Authentication » afin d’activer le service d’authentification de Firebase puis sur « commencer » après la redirection.

Suite à cette action, nous sommes maintenant arrivés sur la page de gestion du service d’authentification et nous avons la possibilité d’ajouter la gestion des adresses e-mail et mot de passe en cliquant sur le bouton « Adresse e-mail/Mot de passe » puis en activant ce service.
