Contexte

Ce cours a pour objectif de vous initier à l’authentification dans une Single-Page Application (SPA) en utilisant Firebase. Dans une SPA, où toute l’application est chargée en une seule page et les interactions se font de manière fluide et réactive, l’authentification doit être gérée de manière efficace et sécurisée. Nous commencerons par explorer les concepts fondamentaux liés à l’authentification dans une SPA, ainsi que les enjeux de sécurité associés. Ensuite, nous nous concentrerons sur la présentation de Firebase et de ses fonctionnalités, en mettant particulièrement l’accent sur Firebase Authentication.
