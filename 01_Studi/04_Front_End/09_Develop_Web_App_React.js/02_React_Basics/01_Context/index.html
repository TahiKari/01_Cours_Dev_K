Contexte

React est une bibliothèque JavaScript populaire qui utilise un langage de balisage appelé JSX et qui est utilisée pour la création d’applications web interactives et dynamiques.

React a été développé par Facebook en 2011 et depuis, elle est devenue une référence incontournable dans le monde du développement web. La raison de cette popularité est due en grande partie à sa facilité d’utilisation et à sa performance élevée.

Contrairement aux frameworks traditionnels, React se concentre sur la création de composants réutilisables et la manipulation dynamique de l’interface utilisateur en réponse aux événements, plutôt que sur la gestion complète de l’application. Cela permet une plus grande flexibilité dans la création de l’interface utilisateur.

Dans ce cours, nous allons étudier le fonctionnement général de React, ses fonctionnalités de base à partir d’un CDN ainsi que ses principes fondamentaux, notamment le concept de SPA (Single-Page-Application) et son approche de rendu côté client (Client-Side-Rendering).

Nous verrons également comment réaliser du SSR (Service-Side-Rendering) avec React.

Il est important de faire la distinction entre la manipulation du DOM (Document Object Model) depuis du JavaScript natif et la manipulation du DOM avec React.

Nous étudierons donc en détail les différences entre ces deux approches.

Ce cours vous permettra de mieux comprendre et aborder React pour l’utiliser efficacement dans vos projets du développement web.
