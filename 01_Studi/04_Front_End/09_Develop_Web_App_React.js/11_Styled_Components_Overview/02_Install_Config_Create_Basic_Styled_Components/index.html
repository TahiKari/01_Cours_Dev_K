Installer la librairie avec npm

Pour utiliser cette librairie, vous devez d’abord l’installer dans votre projet, vous pouvez utiliser npm ou Yarn qui sont deux gestionnaires de paquets populaires. Pour commencer, ouvrez un terminal et accédez au répertoire de votre projet. Exécutez la commande suivante pour installer styled-components (la dernière version en date de la librairie lors de la rédaction de ce cours est la version v6.0.0-rc.2) :

npm install styled-components@latest

Installer la librairie avec Yarn

Si vous préférez utiliser Yarn, ouvrez un terminal, accédez au répertoire de votre projet et exécutez la commande suivante :

yarn add styled-components@latest

Une fois la librairie installée, vous pouvez l’importer dans les fichiers de votre projet où vous souhaitez créer des composants stylisés. Pour importer styled-components, ajoutez la ligne suivante en haut de votre fichier JavaScript comme n’importe quelle importation :

Création de composants stylisés de base

Méthode

Comme nous l’avons vu avant la librairie styled-components utilise le concept de CSS-in-JS, c’est-à-dire que vous pouvez utiliser votre CSS directement dans des variables.

Pour créer un composant stylisé à partir d’un élément HTML, vous pouvez appeler la fonction styled avec le nom de l’élément souhaité. Par exemple, pour créer un composant basé sur un élément <h1>, vous pouvez appeler styled.h1. Ensuite vous définissez les styles CSS entre des accents graves (``).

Création de composants stylisés de base

  • const Title : cela déclare une nouvelle variable constante appelée « Title ». Cette variable va contenir le composant stylisé que nous allons créer avec styled-components.
  • styled.h1 : c’est la fonction fournie par la bibliothèque styled-components qui crée un composant stylisé basé sur un élément HTML <h1>. La fonction styled.h1 attend les styles CSS que vous souhaitez appliquer à l’élément <h1>. Vous pouvez faire de même avec n’importe quelle balise telle que <section>, <div>, <p>, pour ne citer que ceux-là.

Adapter les styles avec les props

Voir video

Afin de personnaliser les styles des composants, nous pouvons transmettre des props à ces composants. Il suffit d’ajouter les props comme attributs lors de l’utilisation du composant. Par exemple, pour un composant Button, nous pouvons lui passer une prop primary pour déterminer si le bouton doit être affiché en tant que bouton principal ou secondaire.

Appliquer des styles conditionnels avec l’interpolation de fonction

Méthode

Pour appliquer des styles conditionnels en fonction des props, nous pouvons utiliser l’interpolation de fonction dans les modèles de chaîne de caractères étiquetés. L’interpolation de fonction est définie par « ${} » et permet d’insérer une expression JavaScript à l’intérieur de la chaîne de caractères.

Dans cet exemple, nous utilisons l’interpolation de fonction pour déterminer la couleur de fond et la couleur de texte du bouton en fonction de la prop primary. Si la prop primary est vraie, le bouton aura un fond bleu et un texte blanc, sinon il aura un fond blanc et un texte bleu. De même, nous utilisons l’interpolation de fonction pour modifier les styles du bouton lorsqu’il est survolé.

Vous remarquerez aussi le « & » avant le :hover, le & est un sélecteur parent dans styled-components. Il représente le composant lui-même dans le contexte des styles. En l’utilisant avec des pseudo-classes ou pseudo-éléments comme le 'hover', vous indiquez que les styles associés doivent être appliqués directement sur le composant.

En utilisant l’interpolation de fonction pour appliquer des styles conditionnels, vous pouvez créer des composants avec styles intégrés qui s’adaptent dynamiquement en fonction des besoins de votre application React.
