Exercice défi

Vous êtes un développeur travaillant sur un projet React pour créer une application web. Votre équipe a décidé d’utiliser la bibliothèque styled-components pour gérer les styles des composants React. Votre tâche est de créer et personnaliser un composant stylisé en utilisant les fonctionnalités de styled-component.

Question

Créez un composant stylisé Button basé sur un élément HTML button avec les propriétés CSS suivantes : couleur de fond bleu, texte blanc et bordure arrondie de 5 pixels. Veuillez fournir uniquement le code du composant stylisé.

Solution

See the Pen Untitled by OpenSpirit (@OpenSpirit) on CodePen.

Dans cet exemple, un composant stylisé nommé Button est créé en utilisant la fonction styled.button de la bibliothèque Styled-components, le composant créé sera basé sur un élément HTML button. Le modèle de chaînes de caractères délimité par les accents graves (``) contient les propriétés CSS pour ce composant demandées dans l’exercice.

Question

Modifiez le composant Button créé précédemment pour qu’il puisse recevoir une prop primary qui, lorsqu’elle est définie à true, change de couleur de fond en vert et la couleur du texte en blanc. Utilisez l’interpolation de fonction pour appliquer ces styles conditionnels en fonction de la valeur de la prop primary. Veuillez fournir uniquement le code du composant stylisé.

Solution

See the Pen Untitled by OpenSpirit (@OpenSpirit) on CodePen.

Dans cet exemple, le composant stylisé Button est modifié pour accepter une prop primary. L’interpolation de fonction est utilisée pour appliquer des styles conditionnels en fonction de la valeur de cette prop. Les propriétés CSS sont définis à l’intérieur des accents graves (``) avec l’interpolation de fonction. Pour la propriété background-color si la prop primary est définie sur true, la couleur de fond du bouton devient verte, sinon elle reste bleue.
