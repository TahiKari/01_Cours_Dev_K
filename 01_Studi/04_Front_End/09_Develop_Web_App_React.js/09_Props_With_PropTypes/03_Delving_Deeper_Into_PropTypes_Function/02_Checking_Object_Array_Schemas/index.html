Vérification des schémas d'objet et de tableau

Lorsque nous avons affaire à des tableaux d'objets, il est important de vérifier non seulement le type de tableau, mais également le type et la forme de chaque objet dans le tableau.

Pour valider un tableau qui contient des objets, nous pouvons utiliser PropTypes.arrayOf() en lui passant en argument un objet représentant le schéma des propriétés attendues pour chaque objet. Cet objet peut être créé avec la méthode PropTypes.shape() en définissant les propriétés attendues et leur type.

Dans cet exemple, nous définissons un schéma d'objet avec la méthode PropTypes.shape() en spécifiant les propriétés id, name et age et leur type attendu. Ensuite, nous utilisons la méthode PropTypes.arrayOf() pour valider le tableau myArrayOfObjects en lui passant le schéma d'objet défini précédemment.

Ainsi, grâce à cette vérification, nous pouvons nous assurer que le tableau passé à notre composant contient des objets conformes au schéma attendu.
