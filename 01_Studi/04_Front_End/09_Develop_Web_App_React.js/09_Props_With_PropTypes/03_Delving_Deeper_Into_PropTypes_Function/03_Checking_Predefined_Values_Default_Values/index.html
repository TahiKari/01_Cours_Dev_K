Vérifier des valeurs pré-définies

Méthode

Lorsque vous définissez une prop, il peut être utile de vérifier que sa valeur correspond à une liste prédéfinie de valeurs. PropTypes.oneOf() est un validateur qui vous permet de définir une liste de valeurs autorisées pour une prop. Par exemple, si vous avez une prop color qui doit être rouge, vert ou bleu, vous pouvez utiliser PropTypes.oneOf() pour valider que la valeur est bien l'une de ces trois options.

Définir des valeurs par défaut

Méthode

Lorsque vous créez un composant React, vous pouvez définir des valeurs par défaut pour certaines de ses props en utilisant la propriété defaultProps. Cela permet de s'assurer que si une prop n'est pas fournie, le composant ne va pas planter ou afficher un comportement inattendu.

Attention

Veillez à ne pas mettre de valeur par défaut sur une prop obligatoire, car cela peut masquer des erreurs de programmation et donner des résultats inattendus, si la valeur par défaut est différente de ce que le composant attend réellement (nom d’utilisateur, identifiant, etc.).
