Installation

Début de l’installation

Nous allons traiter les différentes étapes et procédés pour installer Node.js sur votre appareil en fonction de votre système d’exploitation (Windows, MacOS, Ubuntu).

La principale différence entre l'installation de Node.js sur MacOS et Windows est le type de fichier d'installation utilisé : « .pkg » pour MacOS et « .msi » pour Windows.

Nous allons installer NPM par la même occasion mais nous aurons l’opportunité d’aborder NPM en profondeur dans la suite du cours.

Nous allons suivre ensemble étape par étape le procédé pour l’installation de Node.js et NPM sur Windows, puis sur MacOS et enfin sur Ubuntu.

Récupération du fichier d’installation pour Windows

Méthode

Ouvrez un navigateur web et accédez au site officiel de Node.js.

Notez qu’il est possible d'accéder à deux versions disponibles (LTS et Current) depuis les onglets respectifs.

Sur la page de téléchargement, cliquez sur le bouton « Windows Installer » pour télécharger le fichier d'installation pour Windows.

Installation du fichier téléchargé sur Windows

Méthode

Double-cliquez sur le fichier téléchargé pour ouvrir l'assistant d'installation. Cliquez ensuite sur « Oui » pour autoriser l'application à apporter des modifications à votre ordinateur puis cliquez sur « Next » pour commencer l'installation.

Choisissez l'emplacement d'installation ou utilisez l'emplacement par défaut, puis cliquez sur « Next ».

Laissez actif l’ensemble des composants suggérés pour l’installation, notamment NPM, puis cliquer sur « Next ». Cochez la case « Automatically install the necessary tools » pour installer les outils de compilation, puis cliquez à nouveau sur « Next ».

Cliquez enfin sur « Installer » pour commencer l'installation. Attendez que l'installation soit terminée, puis cliquez sur « Finish » pour quitter l'assistant d'installation.

Node.js et NPM sont désormais installés sur votre système d’exploitation Windows.

Récupération du fichier d’installation pour MacOS

Méthode

Nous allons maintenant suivre ensemble étape par étape le procédé pour l’installation de Node.js et NPM sur MacOS.

Ouvrez un navigateur web et accédez au site officiel de Node.js.

Sur la page de téléchargement, cliquez sur le bouton « macOS Installer » pour télécharger le fichier d'installation pour MacOS.

Installation du fichier téléchargé sur MacOS

Méthode

Double-cliquez sur le fichier téléchargé pour ouvrir l'assistant d'installation. Cliquez sur « Continuer » dans la première fenêtre de l'assistant d'installation. Lisez et acceptez les termes de la licence, puis cliquez sur « Continuer ». Cliquez sur « Installer » pour commencer l'installation. Saisissez votre mot de passe d'administrateur si nécessaire, puis cliquez sur « Installer le logiciel ».

Attendez que l'installation soit terminée, puis cliquez sur « Fermer » pour quitter l'assistant d'installation.

Node.js et NPM sont désormais installés sur votre système d’exploitation MacOS.

Ouverture du terminal et mise à jour système avant installation sur Ubuntu

Méthode

Nous allons maintenant suivre ensemble étape par étape le procédé pour l’installation de Node.js et NPM sur Ubuntu.

Ouvrez un terminal en appuyant sur la combinaison de touche Ctrl + Alt + T. Avant d'installer Node.js, il est recommandé de mettre à jour votre système Ubuntu pour obtenir les dernières mises à jour de sécurité. Vous pouvez le faire en utilisant la commande suivante :

sudo apt update && sudo apt upgrade -y

Installez curl, un outil de transfert de données qui sera utilisé pour télécharger le paquet de Node.js, en tapant la commande suivante :

sudo apt install curl

Pour installer la dernière version de Node.js, ajoutez le PPA de NodeSource à votre système en tapant la commande suivante :

curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - &&\ sudo apt-get install -y nodejs

Ou en root :

curl -fsSL https://deb.nodesource.com/setup_18.x | bash - &&\ apt-get install -y nodejs

Remarque

Cette commande installe Node.js version 18.x. Si vous souhaitez installer une autre version, modifiez simplement le numéro de version dans l'URL ci-dessus. Notez que cette version est compatible avec Ubuntu 20.04 et les versions plus récentes. Les informations relatives à la compatibilité sont disponibles sur ce repository Github.

Ouverture du terminal et vérification de l’installation de Node.js

Méthode

Nous allons maintenant vérifier l’installation de Node.JS et NPM sur votre appareil. Pour cela, ouvrez un terminal.

Voici la méthode pour ouvrir le terminal selon votre système d’exploitation :

  • Sur Windows : appuyez sur la combinaison de touches Windows + R de votre clavier pour ouvrir la boîte de dialogue « Exécuter », puis tapez « cmd » et appuyez sur la touche Entrée. Vous pouvez également ouvrir un terminal en cliquant avec le bouton droit sur le bouton « $Démarrer » de Windows, puis en sélectionnant « v » dans le menu.
  • Sur MacOS : appuyez sur la combinaison de touches Cmd + espace pour ouvrir SpotLight, tapez « Terminal » et appuyez sur la touche Entrée.
  • Sur Ubuntu : appuyez sur la combinaison de touches Ctrl + Alt + T.

Pour vérifier l’installation de Node.js, dans votre terminal tapez la commande suivante : node -v.

Cette commande va afficher la version de Node.js installée sur votre système. Si Node.js est correctement installé, vous devriez voir une version numérique, telle que « v19.7.0 ». Si vous obtenez un message d'erreur, cela signifie que Node.js n'est pas installé sur votre système ou que l'installation a échoué.

Vérification de l’installation de NPM

Méthode

Pour vérifier l’installation de NPM, dans votre terminal tapez la commande suivante : npm -v

Cette commande va afficher la version de NPM installée sur votre système, telle que « 9.5.0 ». Si vous voyez un message d’erreur, NPM n’est pas installé correctement sur votre système.
