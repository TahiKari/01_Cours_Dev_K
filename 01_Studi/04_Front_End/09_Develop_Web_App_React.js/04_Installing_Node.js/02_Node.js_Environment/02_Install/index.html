Installation

Début de l’installation

Nous allons traiter les différentes étapes et procédés pour installer Node.js sur votre appareil en fonction de votre système d’exploitation (Windows, MacOS, Ubuntu).

La principale différence entre l'installation de Node.js sur MacOS et Windows est le type de fichier d'installation utilisé : « .pkg » pour MacOS et « .msi » pour Windows.

Nous allons installer NPM par la même occasion mais nous aurons l’opportunité d’aborder NPM en profondeur dans la suite du cours.
