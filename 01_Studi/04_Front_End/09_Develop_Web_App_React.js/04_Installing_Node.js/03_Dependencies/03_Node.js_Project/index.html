Projet Node.js

Vérification de Node.js et NPM

Méthode

Nous allons maintenant initialiser un projet Node.js. Vous pouvez suivre les étapes ci-dessous.

Assurez-vous que Node.js et NPM sont installés sur votre ordinateur. Vous pouvez mettre en application la méthode vue précédemment dans le cours pour effectuer la vérification.

Initialiser le projet

Méthode

  • Créez un dossier pour votre projet. Vous pouvez le faire en utilisant l'invite de commande ou l'explorateur de fichiers de votre système d'exploitation.
  • Ouvrez ce dossier avec votre éditeur de code puis ouvrez le terminal de votre éditeur de code.
  • Initialisez le projet Node.js en exécutant la commande suivante : npm init -y.

Cette commande crée un fichier package.json avec des valeurs par défaut sans vous demander de remplir des champs. Cette approche est plus rapide car elle vous évite d'avoir à répondre aux questions, mais elle vous donne moins de contrôle sur le contenu de votre fichier package.json.

Si vous exécutez la commande npm init, le terminal vous guide à travers une série de questions pour remplir les champs du fichier package.json, tels que le nom du projet, la version, la description, l'auteur, les dépendances, etc. Cette approche vous permet de personnaliser les options de votre package.json.

Installation d’une dépendance

Méthode

Installez les dépendances dont votre projet a besoin. Vous pouvez le faire en utilisant la commande suivante : npm install <nom de la dépendance>.

Pour l’exemple du cours, nous allons installer la dépendance express, vous pouvez utiliser la commande suivante : npm install express.

Exécuter le projet

Méthode

Exécutez votre projet en utilisant la commande node <nom du fichier>. Par exemple, si votre fichier principal s'appelle « index.js », vous pouvez utiliser la commande suivante : node index.js.

Vous devriez voir s’afficher « Hello Node.js ! » dans la console de votre éditeur de code.
