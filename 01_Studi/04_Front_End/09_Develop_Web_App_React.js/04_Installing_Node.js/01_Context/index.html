Contexte

Dans ce cours nous allons aborder Node.js et ses outils associés, tels que NPM et NPX.

Node.js est un environnement d’exécution côté serveur open-source. Il permet aux développeurs d’écrire des applications en JavaScript côté serveur, en plus des applications côté client traditionnelles. En utilisant Node.js, vous pouvez développer des applications performantes et évolutives en utilisant un seul langage, JavaScript, pour tout votre code.

Dans ce cours, nous allons explorer les bases de l’installation de Node.js sur Windows, Mac et Ubuntu, ainsi que les différentes options de configuration disponibles pour vous permettre de démarrer rapidement. Nous allons également examiner les avantages et les fonctionnalités de NPM et Yarn, les outils de gestion de paquets populaires utilisés pour installer et gérer les dépendances de votre projet Node.js, ainsi que NPX, un exécutable couramment utilisé pour exécuter des commandes en utilisant les packages téléchargés via NPM.
