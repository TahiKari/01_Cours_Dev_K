Exercice défi

Supposons que vous ayez besoin de créer une application web. Vous avez décidé d'utiliser Node.js pour la partie back-end de l'application.

Question

Vous devez créer un dossier et initialisez un projet Node.js en créant un fichier package.json avec NPM. Comment procédez-vous ?

Solution

Pour créer un dossier, vous pouvez le faire manuellement via votre gestionnaire de fichier ou vous pouvez utiliser la commande mkdir dans le terminal ou l'invite de commande, par exemple :

mkdir mon-projet

Ensuite, accédez au dossier que vous venez de créer et utilisez la commande npm init pour initialiser un nouveau projet Node.js et créer un fichier package.json :

cd mon-projet

npm init

Vous pouvez suivre les instructions de l'assistant pour renseigner les informations relatives à votre projet ou bien choisir d’utiliser la commande npm init -y pour générer un fichier package.json par défaut sans avoir à renseigner les informations relatives au projet.

Question

Installez une dépendance à votre projet (par exemple Express) et utilisez NPM pour vérifier l'installation et la version de cette dépendance.

Solution

Pour installer une dépendance, vous pouvez utiliser la commande npm install suivie du nom de la dépendance, par exemple :

npm install express

Cela va télécharger et installer la dernière version d'Express et ajouter une entrée correspondante dans votre fichier package.json.

Pour vérifier que la dépendance est bien installée et connaître sa version, vous pouvez utiliser la commande npm list. Cela va afficher toutes les dépendances de votre projet, avec leur nom et leur version.
