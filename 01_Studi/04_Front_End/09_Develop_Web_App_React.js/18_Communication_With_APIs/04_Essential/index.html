Essentiel

Dans ce cours, nous avons appris comment interagir avec une API web tierce dans une application React, à l’aide de l’API Fetch ou de la librairie Axios.

Nous avons vu ce qu’est une API et quelles API sont utilisées dans le contexte de la programmation web.

Nous avons vu comment envoyer une requête à une API web tierce, avec notamment l’exemple de l’API geo.api.gouv qui permet de récupérer des données des villes françaises. Nous avons vu comment afficher ces données de telle manière dans notre application React, et comment combiner une requête avec l’entrée d’un utilisateur dans un formulaire, dans notre premier exemple pour déterminer le département des villes à afficher.

Nous avons également vu comment Axios facilite la rédaction de requêtes HTTP et comment utiliser certaines de ses fonctionnalités, par exemple la gestion des erreurs, ou la création d’une instance Axios qui intègre des paramètres préconfigurés.

Enfin, nous avons vu à quel point il est aisé et rapide de mettre en place une application React qui interagit simplement avec une API et les affiche très rapidement dès qu’elles sont récupérées, sans qu’il soit nécessaire d’écrire des lignes et des lignes de code complexes.
