Pourquoi D3.js ?

D3.js, pour « Data-Driven Document », est une librairie graphique basée sur l’affichage de données interactive et dynamique.

Cette librairie combine JavaScript et essentiellement la technologie SVG, ou « Scalable Vector Graphics », pour mettre à la disposition des utilisateurs une API rendant accessible la création de graphiques détaillés et interactifs, notamment grâce à des fonctionnalités de transition et d’animation. Son aspect bas-niveau la rend notamment flexible et compatible avec toutes sortes d'algorithmes, le seul pivot de ce framework est la manipulation de données.
