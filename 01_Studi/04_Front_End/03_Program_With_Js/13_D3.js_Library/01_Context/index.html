Contexte

JavaScript est un langage de programmation polyvalent couramment utilisé pour créer des applications interactives sur le Web. En combinaison avec le langage de balisage HTML et les feuilles de style CSS, il permet de rendre les pages web dynamiques et interactives. Une bibliothèque JavaScript puissante et populaire pour la manipulation et la visualisation de données sur le Web est D3.js.

D3.js (Data-Driven Documents) est une bibliothèque JavaScript qui offre un large éventail de fonctionnalités pour la création de visualisations de données interactives. Elle permet de lier des données chargées de différentes sources, telles que des fichiers CSV, JSON ou même des API en lignes, et de les lier à des éléments HTML existants manipulés de manière dynamique en utilisant les principes du DOM. L’usage de cette librairie se concentre sur la création de graphiques, de diagrammes, de cartes et d'autres visualisations de données.
