L’essentiel

Dans ce cours, nous avons réalisé un formulaire de contact dynamique avec JavaScript. JavaScript permet de facilement accéder aux éléments des pages HTML via le Document Object Model, ce qui constitue un véritable atout pour les développeurs web. Il contient par ailleurs des classes prédéfinies comme FormData qui permet de récupérer facilement les données des champs d’un formulaire. JavaScript va donc être essentiel pour ajouter du dynamisme et de l’interactivité à un site Internet.
