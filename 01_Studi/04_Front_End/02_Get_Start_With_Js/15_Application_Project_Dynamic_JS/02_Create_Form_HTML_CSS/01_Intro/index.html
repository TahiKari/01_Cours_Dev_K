Introduction

Objectifs de la partie

  • Maîtriser la création d’une architecture d’un formulaire HTML.
  • Maîtriser l’édition de styles en CSS.

Contexte

Pour un développeur de sites web, la maîtrise des standards HTML/CSS est fondamentale. Dans cette partie, nous réaliserons un formulaire en utilisant ces deux technologies. Nous créerons des classes et des éléments qui seront plus tard utilisés dans ns scripts JS. Cette partie va donc nous permettre de poser les fondements de notre formulaire.

Présentation du visuel du formulaire

Nous allons réaliser un formulaire de contact dynamique que nous pourrions trouver sur un site « vitrine » d’un professionnel. Ce formulaire permettra de recueillir les informations suivantes :

  • Le nom de l’utilisateur,
  • Le nom de l’utilisateur,
  • La société de l’utilisateur (uniquement si c’est un professionnel),
  • L’objet de son message,
  • Son adresse email dans le cas où l’objet est « demande de contact »,
  • Son message.
