Introduction

Objectifs de la partie

  • Maîtriser les principes de base de la définition d’une classe en JS.
  • Utiliser les moyens d’accès au Document Object Model.

Contexte

Pour un développeur qui crée un formulaire, il va être indispensable de rajouter de l’interactivité et des moyens de traiter les données avec un langage de programmation. JavaScript va remplir ce rôle, du côté client. Pour cela, nous allons faire appel à des notions fondamentales de Programmation Orientée Objet et nous allons manipuler les éléments du Document Object Model, pour ainsi modifier l’affichage de notre page.
