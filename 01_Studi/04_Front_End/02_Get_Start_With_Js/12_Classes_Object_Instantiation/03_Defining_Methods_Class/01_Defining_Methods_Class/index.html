Définir une méthode statique

Méthode

En JavaScript, définir une méthode statique revient à définir une propriété statique, mais sur une fonction. Pour cela, nous allons utiliser le mot clé static. Plutôt que d’utiliser la syntaxe maMethode = function() {}, nous pouvons utiliser le raccourci maMethode() {}. Nous n’avons donc pas besoin d’utiliser le mot function.
